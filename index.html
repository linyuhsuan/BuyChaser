關鍵技術

Python

利用Python將在直播所聽到的語音訊息轉換成文字檔，將所聽到的關鍵字詞自動擷取下來，進而進行分析比對。

PHP

PHP提供免費、開發環境齊全、與資料庫連結方便以及良好的執行效能等優點，因此BuyChaser利用PHP去撰寫系統。

Jieba

將直播當中的話語音轉成文字，再利用語意分析比對模組，透過Jieba進行斷詞，將直播的關鍵字詞與必買清單去做比對。

Facebook messenger API

利用Facebook提供的API傳寫Chatbot，能與使用者及時互動、自動回覆，幫助消費者找到想要的商品，此外，也能將直播當中的留言+1同步。
